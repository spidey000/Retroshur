  Left Analog Stick: WASD keys (movement)
  Right Analog Stick: Mouse Movement (camera control, cursor control)
  Directional Pad:  Up = Mouse Wheel Up (zoom in)
                           Down = Mouse Wheel Down (zoom out)
                           Left = Q Key (unsheathe melee weapon)
                           Right = E Key (unsheathe ranged weapon)
  Button 1:  Left Shift (ready magic, absorb exp. orbs)
  Button 2:  F3 Key (context action 3)
  Button 3:  Caps Lock (activate/deactivate stealth mode)
  Button 4:  Tab (interact)
  Button 5:  Space Key (target lock)
  Button 6:  Middle Mouse Button (block, evade)
  Button 7:  Left Mouse Button (attack, cast left magic, select)
  Button 8:  Right Mouse Button (run, power attack, first-person mode (bow), cast right magic)
  Button 9:  G Key (guild teleport)
  Button 10:  Escape Key (menu)
  Button 11:  F1 Key (context action 1)
  Button 12:  F2 Key (context action 2)
